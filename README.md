## 📂 Projects Overview

### 🔹 Level 1 – Café Website ☕
A responsive café website focused on UI design and layout structuring.

**Features:**
- Fully responsive design
- Home, About, Menu, Gallery, Contact pages
- Interactive menu (+ / - functionality)
- Total price calculation using JavaScript

---

### 🔹 Level 2 – Quiz Application 🧠
An interactive quiz app with dynamic logic and real-time feedback.

**Features:**
- Multiple-choice questions
- Timer-based quiz
- Score calculation
- Result summary (correct/wrong/score)
- Responsive UI

---

### 🔹 Level 3 – E-Commerce Website 🛒
A real-world inspired shopping website with complete MERN stack architecture.

**Features:**
- Product listing & details with Reviews & Ratings
- Add to cart & Wishlist functionality
- Full Checkout flow with Address persistence
- Cancel/Delete Order features & Order History
- Responsive Flipkart-Style UI
